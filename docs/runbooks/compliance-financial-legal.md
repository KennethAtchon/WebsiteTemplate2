## Item 57 — Financial Compliance (PCI DSS, Tax, Audit Trail)

### PCI DSS

**Status: In scope — Stripe handles PCI compliance.**

YourApp uses Stripe.js and the Stripe Customer Portal for all payment flows. Card data never touches YourApp's servers.

- **PCI scope:** SAQ A (the most minimal scope; card data only flows through Stripe's domain)
- **Action required:** Annually complete Stripe's [SAQ A self-assessment](https://stripe.com/guides/pci-compliance) (5 minutes)
- **Stripe's PCI certificate:** Available in Stripe Dashboard → Settings → Compliance

### Tax Compliance

- Stripe Tax handles VAT/sales tax calculation and collection where configured
- **Action:** Enable Stripe Tax in Stripe Dashboard → Tax → Overview for applicable regions
- Revenue receipts are automatically generated by Stripe for each transaction

### Audit Trail

The app maintains an audit trail via:
- `FeatureUsage` table — all calculator uses with timestamps
- `Order` table — all payment events with Stripe session IDs
- Stripe Dashboard — complete payment history and invoices
- Railway logs — API access logs with timestamps

**Retention:** Financial records retained for 7 years per UK/EU accounting requirements (see `data-retention-policy.md`).
